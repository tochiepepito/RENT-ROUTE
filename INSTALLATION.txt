## 🚀 How to Use

### Step 1: Test Registration

1. Open your browser
2. Navigate to: `html/template/register.html`
3. Fill in the form:
   - **Role**: Select "Renter" or "Car Owner"
   - **Username**: Any name (must be unique)
   - **Email**: Valid email address
   - **Password**: Any password (min 6 chars)
4. Click "Sign Up"
5. ✅ You'll be logged in and redirected

### Step 2: Verify in Database

1. Open: `database/accounts.json`
2. You should see your new account with:
   - Unique user_id
   - Hashed password
   - Selected role
   - Creation timestamp

### Step 3: Test Login

1. Logout by going to: `backend/logout.php`
2. Navigate to: `html/template/login.html`
3. Enter your email and password
4. Click "Sign In"
5. ✅ You'll be logged in again

### Step 4: Use in Your Pages

Add authentication to any PHP page:

```php
<?php
require_once '../../backend/auth.php';

// Check if logged in
Auth::requireLogin();

// Get user info
echo Auth::getUserEmail();
echo Auth::getUserRole();
?>

<!-- Page content here -->
```

## 💡 Usage Examples

### Protect a Page
```php
<?php
require_once '../../backend/auth.php';
Auth::requireLogin();
// Page content
?>
```

### Check Specific Role
```php
<?php
require_once '../../backend/auth.php';
if (Auth::hasRole('renter')) {
    echo "Renter content";
}
?>
```

### Restrict to Role
```php
<?php
require_once '../../backend/auth.php';
Auth::requireRole('car_owner');
// Only car owners can access
?>
```

### Get User Information
```php
<?php
require_once '../../backend/auth.php';
$email = Auth::getUserEmail();
$role = Auth::getUserRole();
$user_id = Auth::getUserId();
?>
```

## 🆘 Troubleshooting

### Issue: "Permission denied" when registering
**Solution:** Make database folder writable
```bash
chmod 755 database/
chmod 644 database/accounts.json
```

### Issue: Sessions not working
**Solution:** Verify PHP session configuration in php.ini

### Issue: Password verification fails
**Solution:** Ensure PHP 7.4+ (BCrypt is built-in)

### Issue: Page redirects to login
**Solution:** Verify you called `Auth::requireLogin();` correctly

### Issue: Can't access test.php
**Solution:** Check file path and server access permissions
