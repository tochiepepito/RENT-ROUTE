❌ TROUBLESHOOTING:

Problem: "Database connection failed"
Solution: 
  1. Start MySQL in XAMPP
  2. Check config.php credentials
  3. Run setup wizard again

Problem: Login not working
Solution:
  1. Ensure account was created in registration
  2. Check email and password are correct
  3. Make sure role matches registration role
  4. Check browser console for errors

Problem: Redirect not working
Solution:
  1. Ensure user-dashboard.html exists
  2. Ensure admin/index.html exists
  3. Clear browser cache (Ctrl+Shift+Delete)
  4. Check browser console for errors
